# 产品想法

## 文档用途
- 只记录还在讨论中的产品方向和想法。
- 这里的内容不等于已经确认立项，也不等于本轮一定会做。
- 只有当某个想法被确认进入开发，才迁移到 `02-迭代计划.md`。

## 首页与推荐

### 首页是否保留“今日”
- 当前讨论方向：保留首页。
- 首页承担自动推荐入口。
- 新学习建议当前阶段可以暂时不做。

### 自动推荐
- 用户进入首页后，系统自动生成推荐。
- 每次进入都获取最新结果，不做缓存。

### 复习建议
- 复习建议改为规则驱动，不依赖大模型。
- 已形成一版更细的排序 PRD，见 `docs/prd/06-今日推荐排序PRD.md`。
- 这套排序结果同时应用于首页自动推荐和诗库列表排序。

### 新学习推荐
- 当前讨论倾向：暂时不做。
- 如后续恢复，可继续保留大模型能力。
- 候选方向：
  - 用户需求驱动：如果用户想背课本诗词，则从课本需求池中推荐。
  - 关联推荐：根据用户背过的诗推测偏好，推荐相关主题、作者、意象、风格的诗词。

### 自定义推荐
- 允许用户输入自己想背的类型、主题或方向。
- 系统根据输入生成个性化推荐。

## 名句模块
- 增加诗词名句列表。
- 支持按类型分类浏览。
- 首页或其他入口可按一定规则推荐名句，用于激发兴趣。

## UI 与信息架构

### 底部 Tab 简化
- 去掉“数据”Tab。
- “模型”Tab 是否保留，暂未完全决定。
  - 当前判断：由于默认模型已经可用，可以弱化模型配置入口。
  - 但如果默认模型异常，仍可能需要保留一个维护入口。
- 底部 Tab 结构后续仍需要继续收敛。

### 诗库页改版
- 诗库页不再默认展示大块“口语输入”区域。
- 在诗库列表右上角提供一个“+”入口。
- `想背` 按钮放在诗库列表标题行右侧。`r`n- 该按钮为可切换按钮，点一次标记想背，再点一次取消，状态反馈要清晰。
- `建议复习` 标记展示在列表 `浏览次数` 右侧，和现有信息区保持一致。
- 点击“+”后，再进入或弹出口语输入界面。
