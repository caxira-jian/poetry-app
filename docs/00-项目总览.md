# 项目总览

## 项目名称
- 古诗背诵助手

## 项目目标
- 做一个手机优先的古诗背诵 Web 应用。
- 支持本地诗库、背诵记录、口语输入解析、复习推荐。

## 当前线上地址
- [poetry-app-chi.vercel.app](https://poetry-app-chi.vercel.app/)

## 技术栈
- 前端：Vue 3 + Vite
- 存储：IndexedDB
- 部署：Vercel
- 大模型接入：服务端默认 API 代理 + 浏览器本地自定义 API

## 当前迭代重点
- 让诗库浏览、详情页、打卡和推荐形成更完整的学习闭环。
- 提升移动端体验，尤其是详情页的返回行为与正文阅读体验。

## 当前已确认的重要机制
- 详情页进入即计 1 次浏览。
- 详情页支持今日背诵打卡，同一天只允许 1 次。
- 安卓/iOS 的返回手势或系统返回，在详情页时优先回到应用内上一层，而不是直接离开网页。
