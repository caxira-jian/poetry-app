# 发布清单

## 发布前检查
- `npm run build` 通过
- 核心页面可打开
- 核心路径手动检查
- 新增字段或存储变更已考虑旧数据兼容
- 新增环境变量已写入文档
- 重要行为变更已更新开发日志
- 产品现状文档已同步

## 建议手测路径
- 诗库初始化 / 导入
- 诗库口语输入解析与确认入库
- 诗词详情页打开与返回
- 背诵打卡
- 浏览次数变化
- 今日推荐
- 默认 API 连通性测试

## 发布后验收
- 线上首页可打开
- 默认 API 可正常调用
- 详情页可打开
- 详情页返回不会直接退出网页
- 打卡按钮行为符合预期
- 控制台和 Vercel 无明显错误

## 发现问题后的动作
- 先记录现象
- 标记影响范围
- 建 issue
- 再决定 hotfix 还是放入下一轮
